Create crypto widgets for 250+ coins which come with this Elementor addon and Use this advanced plugin in simple steps on Elementor.

There are four layouts in this widget , where you can show your coins in different styles.

<ol>
<li><strong>Coins Price Card</strong>
Show coins in beautiful card layout with multiple dynamic attribute feature,where you can easily customize layout of your card.
You can hide show coin symbol, high/low,1h changes, 24h changes,7d changes, 30d changes, rank, market cap from these settings,
 you can also enable disable number formatting from these settings.
</li>
<li><strong>Coins Price Label</strong>
Display coin price label inside your page content using this style.You can hide show coin 24h changes from these settings.
</li>
<li><strong>Coins List</strong>
In this layout you can show Top coins from the given list or you can add your favorite custom coins by selecting the custom option from the coin list.
 In this layout there's a option to show 7 day data in chart format, by default its enabled in this layout but you can enable or disable this chart from show hide graph option.You can hide/show coin 24h changes and graph from these settings.
</li>
<li><strong>Top gainer/loser</strong>
In top gainer loser list you can display 24 hour changes of coin from 250 coins list,in gainer you will see top most increased coin value from the last 24 hours,

in loser list it will display top most decrease coins value from the last 24 hours.You can hide/show coin 24h changes from these settings.

</li>

</ol>

== Installation ==
#### Install From WordPress Admin Panel

Open WordPress admin panel and go to Plugins >> Add New and Search **“Cryptocurrency Widgets For Elementor”** at here, You will find our plugin **“by Cool Plugins”**. Click on Install button in front of this and then activate the plugin.

#### How To Create Crypto Top gainer/losers list Widget?

After installation & plugin activation, create a page and edit page with elementor, in elementor search bar type Cryptocurrency Widget. From the search result drag Cryptocurrency Widget to your desired location on the page. By default it will add Card layout, you change layout from widget settings panel. Also can style the individual widget from style tab colors options according to your requirement.

== Frequently Asked Questions ==
#### Can I display real time live price changes?

We are using CoinGecko api in our free version of crypto price widget, it can show only current price but doesn’t support real time price updates.

#### Can I choose my local currency other than USD($)?

Yes, you can display crypto coins price in your local currency. we have integrated 32 fiat currencies:- **USD, GBP, EUR, INR, JPY, CNY, ILS, KRW, RUB, DKK, PLN, AUD, BRL, MXN, SEK, CAD, HKD, MYR, SGD, CHF, HUF, NOK, THB, CLP, IDR, NZD, TRY, PHP, TWD, CZK, PKR, ZAR**.

#### Are you using any third party APIs?

YES, we are using coingecko.com apis in our free version. So plugin functionality totally depends on third party apis. (We can’t provide any support if there is any downtime in third party apis)
